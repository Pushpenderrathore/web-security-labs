### Repository Overview

This repository contains practical lab assignments for learning Web Technologies and Web Application Security. It covers frontend and backend development fundamentals, real-world web server configurations, and essential cybersecurity techniques focused on identifying and exploiting vulnerabilities in a controlled and ethical environment.

> All work is structured by assignment and includes configuration notes, testing reports, scripts, and exploitation documentation.
